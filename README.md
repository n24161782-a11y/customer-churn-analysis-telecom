# Customer Churn Analysis for Telecom Industry

## Project Overview
Customer churn is a major challenge for telecom companies. 
This project builds a machine learning model to predict whether a customer will leave the telecom service.

## Objectives
- Identify customers likely to churn
- Analyze factors affecting churn
- Provide recommendations to reduce churn

## Tools & Technologies
Python
Pandas
Scikit-learn
Seaborn
Matplotlib
Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation

## Machine Learning Model
- Logistic Regression
- Random Forest Classifier

## Model Evaluation
Accuracy
Confusion Matrix
Precision & Recall

## Key Insights
- Customers with high monthly charges have higher churn rate
- Contract type strongly influences churn
- Customers with shorter tenure churn more

## Business Recommendations
- Provide discounts for high monthly charge users
- Offer loyalty rewards for long-term customers
- Improve support for new customers
